In order to change your CSS password, please follow these instructions:


    • Check the checkbox next to "Change Password".

    • Enter your current password in the "Password" box.

    • Enter your new password twice.
      • - once in the "New Password box.
      - once in the "Confirm New Password" box.
    • Rules for passwords:
      - Length of 8 characters.
      - The password must contain 3 out of 4 of the following types of characters:
        • English lowercase letters
        • English uppercase letters
        • Arabic numerals(0,1,2,...9)
        • Non alphanumeric special characters:
           (No : and <space>) ! @ # $ % ^ & * () + { } | " < > ? ' . / [ ] \ - = ~ `

In order to reset your CSS password, please follow these instructions:


    • Contact the IT NSD (National Service Desk) at 855-NSD-HELP (855-673-4357)

    • Select the option for "NSD Philadelphia and VBA Support" and then select the option for "Password Reset"

    • After validating your contact information with the service desk technician, specify:
      - that you need a CSS password reset.
      - which VA application you are logging into.
